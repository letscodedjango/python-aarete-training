What is Function?

Function is nothing but a block of code that helps to perform operations on data.

Another way of defining the same, function is a set of instructions or lines of code that helps to perform repetative task or same operation on multiple data

Types of Functions?

There are 2 types of functions in Python

  • Built-in Functions

    • print() / len() / sqrt() / cbrt() and many more ....

  • User-defined Functions

What is Method?

Method is nothing but a block of code that helps to perform operations on data.

Another way of defining the same, method is a set of instructions or lines of code that helps to perform repetative task or same operation on multiple data

When we wrap function inside class then that function become method

Types of Methods?

There are 3 types of Methods in Python

  • Instance Method

    • The instance method takes self as first argument.

    Instance method can access and modify the object (instance of class) state.

    No decorators are needed for defining instance method


  • Class Method

    • The class method takes cls (class) as first argument.

    Class method can access and modify the class state.

    The class method takes the class as parameter to know about the state of that class as first argument.

    @classmethod decorator is used here.


  • Static Method

    • The static method does not take any specific parameter.

    Static Method cannot access or modify the class state.

    Static methods do not know about class state. These methods are used to do some utility tasks by taking some parameters.

    @staticmethod decorator is used here.

    staticmethod() function can also be used to covert any function to static.

4 DO's for Functions/Methods in Python

  • Always use def keyword to define any function or method

  • Always suffix name of function or method with paranthesis ()

  • Always need to call function or method by its name to execute

  • Always specify paranthesis () with name at the time of calling function
