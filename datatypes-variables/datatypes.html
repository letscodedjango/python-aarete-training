Here, we gonna discuss regarding the python datatypes and variables.

What is Datatype?

Data Type represents the type of data a variable can store.

Types of Datatype?

There are various types of Data types in Python

  • Integer (int)
  • Float (float)
  • Boolean (bool)
  • String (str)

It represents/tells how much memory a particular variable should hold within the memory and what type of value a particular variable is gonna hold.

What is Variable?

Variables are nothing but a container that can hold data.

Technically, Variables are nothing but a memory location within our RAM to hold data that can be used in our program.

Types of Variables?

There are various types of Variables in Python

  • Global Variable

  • Instance Variable

  • Local Variable

3 DO's for variables in Python

  • Variable Declaration
    • int x
  • Variable Intialization
    • x = 50.89
  • Variable Definition
    • int x = 56.99
